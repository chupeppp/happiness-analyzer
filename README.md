# Happiness Analyzer

## Описание проекта
Проект представляет собой консольное Java-приложение для анализа данных отчета World Happiness Report 2015.
Приложение выполняет парсинг CSV-файла, сохраняет данные в базу данных SQLite, выполняет аналитические SQL-запросы
и визуализирует числовые показатели в виде диаграмм.

---

## Цель работы
Целью проекта является:
- освоение работы с CSV-файлами;
- проектирование объектной модели данных;
- нормализация данных и работа с SQLite;
- выполнение аналитических SQL-запросов;
- визуализация данных;
- оформление проекта в виде GitHub-репозитория.

---

## Используемые технологии
- Java 17+
- Maven
- OpenCSV
- SQLite (sqlite-jdbc)
- JFreeChart

---

## Этапы выполнения работы

### 1. Анализ CSV-файла
Исходный файл `happiness_data_2015.csv` содержит данные по странам:
- регион
- индекс счастья
- экономический показатель (GDP per Capita)
- здоровье
- свобода
- доверие к правительству
- щедрость

### 2. Проектирование объектной модели
Были созданы классы:
- `Country` - хранит показатели страны
- `Region` - хранит название региона

### 3. Парсинг CSV
С помощью библиотеки OpenCSV данные были считаны и преобразованы в объекты Java.
Регионы сохраняются без дублирования.

### 4. Создание базы данных
Используется SQLite.
Созданы таблицы:
- `regions`
- `countries`

База данных приведена к третьей нормальной форме.

### 5. Заполнение базы данных
Все объекты Country и Region сохранены в БД.

## SQL-запросы

### Задание 1
Построение графика по показателю экономики, объединенного по странам.
(Используется топ-10 стран по GDP per Capita)

### Задание 2
Определение страны с наибольшим показателем экономики
среди регионов:
- Latin America and Caribbean
- Eastern Asia

Результат выводится в консоль.

### Задание 3
Поиск страны с наиболее «средними» показателями
для регионов:
- Western Europe
- North America

Под средними понимается минимальное суммарное отклонение от средних значений всех показателей.
