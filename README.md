# Анализ счастья стран (2015)

Семестровый проект по обработке данных о счастье стран за 2015 год.

## Выполненные задачи

1. Реализованы классы `Country` и `Region` для хранения данных
2. Выполнен парсинг CSV-файла с данными
3. Создана SQLite база данных по 3НФ:
   - Таблица `regions`
   - Таблица `countries`
4. Реализованы SQL-запросы:
   - Топ-10 стран по показателю экономики (GDP per Capita)
   - Поиск страны из топ-3 по экономике в регионах Latin America/Caribbean и Eastern Asia
   - Поиск страны со "средними показателями" в Western Europe и North America
5. Результаты визуализированы в виде диаграмм

## Как запустить

1. Установите JDK 11+
2. Соберите проект: `mvn clean package`
3. Запустите:  
   `java -jar target/happiness-analyzer-1.0-SNAPSHOT-jar-with-dependencies.jar`
